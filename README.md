## AYSSOFT TASK

### Kurulum

- Projeyi klonlayın.

git clone  ```  https://github.com/sertacgltkn/ayssoft-task.git ```

- Projeyi çalıştırın.

``` cd ayssoft-task ```

``` npm install ```

``` npx expo start ```

##### NOT Projeyi android cihazınızda çalıştırmak için;

``` npm install -g expo-cli ``` kurulu olmalıdır.


### Kullanılan Teknolojiler

- React Native
- Expo
- React Navigation
- React Native Async Storage
- Axios
